myCalendarMap module¶

myCalendarMap.date_heatmap(series, start=None, end=None, mean=False, ax=None, **kwargs)¶

Plot a calendar heatmap given a datetime series.

Parameters

  • series (pd.Series) – A series of numeric values with a datetime index. Values occurring on the same day are combined by sum.

  • start (Any) – The first day to be considered in the plot. The value can be anything accepted by pandas.to_datetime(). The default is the earliest date in the data.

  • end (Any) – The last day to be considered in the plot. The value can be anything accepted by pandas.to_datetime(). The default is the latest date in the data.

  • mean (bool) – Combine values occurring on the same day by mean instead of sum.

  • ax (matplotlib.Axes or None) – The axes on which to draw the heatmap. The default is the current axes in the :module:`~matplotlib.pyplot` API.

  • **kwargs – Forwarded to pcolormesh() for drawing the heatmap.

Returns

The axes on which the heatmap was drawn. This is set as the current

axes in the ~matplotlib.pyplot API.

Return type

matplotlib.collections.Axes
