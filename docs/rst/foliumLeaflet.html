foliumLeaflet module¶

foliumLeaflet.drawSessions(rSessions, mapLinesArgs)¶

The main draw leaflet function to draw one or more sessions. :param rSessions: List of one or more running sessions. This is what myStravaImportData returns. :type rSessions: list :param mapLinesArgs: A dictionary for controlling multiple drawing functions in leaflet map. See examples below :type mapLinesArgs: dict

Returns

Returns a folium map object

Return type

myMapSessions (obj)

Example

This is an example on how to use it:

sessionsToDraw = myData[:10]
myMapLinesArgs = {
    'tiles': 'Cartodb Positron',
    'zoom': 3,
    'lineWeight': 3,
    'lineCol':'#3388ff'
    }

drawMap = flmap.drawSessions(sessionsToDraw,myMapLinesArgs)
drawMap
